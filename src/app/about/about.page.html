Botão de Emergência

O presente projeto propõe o desenvolvimento de um Sistema de Alerta para Pacientes com Doenças Crônicas, voltado para situações de emergência onde o paciente necessita de atendimento rápido por parte de um cuidador.

O sistema utiliza uma integração entre Arduino Uno, módulo Bluetooth, aplicativo móvel (Ionic) e plataforma Firebase, possibilitando o acionamento de alarmes locais e notificações remotas.
